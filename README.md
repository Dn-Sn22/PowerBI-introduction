# PowerBI-introduction



## Описание
Этот репозиторий содержит PowerBI файлы включая проект и ознакомительные работы.

---

## Цель проекта
Цель — создать рабочий Power BI дашборд, который:
- визуализирует ключевые метрики,
- показывает важные бизнес-инсайты,
- демонстрирует навыки работы с данными.

## Этапы работы

### 1. Подготовка данных (Power Query)  
В Power Query:
- удаление лишних строк/столбцов,
- исправление типов данных,
- объединение таблиц,
- создание вычисляемых столбцов

### 3. Модель данных  
- joins, relationships таблиц, настроивание ключей и связей
- создание мер.

### 4. Визуализация  
Создал интерактивные отчёты с:
- Диаграмма дерева
- Гистограмма 
- Линейчатая диаграмма
- Карточки
- Датчик
- Кольцевой график
